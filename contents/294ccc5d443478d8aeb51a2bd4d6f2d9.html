Alterar Vazão

UMLUseCase
Projeto2 :: Use Case Diagram :: Alterar Vazão

Description

Fluxo principal

  1. O operador se dirige a um dos componentes do sistema de biodiesel

  2. O operador altera o valor de vazão desse componente

Fluxo Secundário O valor da vazão informado pelo operador excede o valor máximo em l/s ou está abaixo do nível mínimo.

  1. O operador se dirige a um dos componentes do sistema de biodiesel

  2. O operador tenta alterar o valor de vazão máxima/mínimo, porém o sistema não permite a adulteração do valor.
