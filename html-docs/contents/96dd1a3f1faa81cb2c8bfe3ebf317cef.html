Lavagem

UMLClass
Projeto2 :: Class Diagram :: Lavagem

Description

Esta classe representa a entidade da lavagem.

Atributos:

  • Vazão Limite: Representa a vazão limite da lavagem em l/s.

Métodos:

  • ReceberÁgua(): Ato da lavagem de receber água de outros componentes do sistema.

  • ReceberMistura(): Ato da lavagem de receber a mistura de outros componentes do sistema.

  • ProcessaLavagem(): Ato de processar lavagem

  • Emulsiona(): Ato de emulssionar

  • TransferirParaTanque(): Ato do decantador de transferir a mistura para um tanque.
