Projeto2 :: Class Diagram :: Reator

Description

Esta classe representa a entidade do reator.

Atributos:

  • Capacidade Máxima: Representa a capacidade máxima de armazenamento do reator.

  • Capacidade Utiliza: Representa a capacidade atualmente ocupada pelo reator.

  • Vazão: Representa a vazão reator em l/s.

  • Velocidade de Processamento: Representa a velocidade média de processamento da mistura do reator em l/s.

Métodos:

  • ProcessaMistura(): Ato do reator de processar a mistura.

  • TransfereParaDecantador(): Ato do reator de transferir a mistura para o decantador.
