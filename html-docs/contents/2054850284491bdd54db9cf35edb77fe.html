Operador

UMLClass
Projeto2 :: Class Diagram :: Operador

Description

Esta classe representa a entidade do operador.

Métodos:

  • Abastecer(): Ato do operador abastecer algum componente envolvido no sistema.

  • AlterarValoresEntrega(): Ato do operador de modificar o valor de entrega máximo/mínimo de algum componente envolvido no sistema.

  • AlterarVazão(): Ato do operador de modificar o valor máximo/mínimo de vazão de algum componente envolvido no sistema.

  • AprovarAlterações(): Ato do operador de aprovar o relatório com as alterações informadas

  • CadastrarNívelSubstâncias(): Ato do operador de cadastrar o nível final das substâncias.

  • VerificarNívelSubstâncias(): Ato do operador de verificar o nível final das substâncias

  • VerificarNívelSubstâncias(): Ato do operador de verificar os valores de entrega.

  • AbrirVálvulas(): Ato do operador de abrir as válvulas.
