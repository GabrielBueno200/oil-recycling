Description

Fluxo principal

  1. O operador decide alterar as entregas

  2. O operador altera o valor de uma das entregas

Fluxo Secundário O valor informado pelo operador excede o valor máximo de entregas ou está abaixo do nível mínimo.

  1. O operador se dirige a um dos componentes do sistema de biodiesel

  2. O operador tenta alterar o valor de entrega máxima/mínimo, porém o sistema não permite a adulteração do valor.
